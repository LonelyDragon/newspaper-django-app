## About
Учебный проект с книги [Django for Beginners](https://djangoforbeginners.com/) развернут на [Heroku](https://django-blog-dfb20.herokuapp.com/)
Немного доделал:
1. Доработана система комментариев:
'''
Вывод всех комментариев под каждым постом.
Добавлена форма комментирования.
Реализовано ограничение списка доступных операций над постом пользователю.
'''
2. Изменена сортировка постов по дате (новые -> старые)
3. Добавлено e-mail оповещение для сброса пароля

_Ключи проекта вынесены в отдельный файл конфига_
## Requirements
Используется:
- Django 3.0.1
- django-crispy-forms 1.8.1
- whitenoise 4.1.4
- gunicorn 19.9.0
- Python 3.7
